Supported CVD Technologies (9 variants):
    ✓ APCVD   (Atmospheric Pressure)
    ✓ LPCVD   (Low Pressure)
    ✓ UHVCVD  (Ultra-High Vacuum)
    ✓ PECVD   (Plasma-Enhanced)
    ✓ HDP-CVD (High-Density Plasma)
    ✓ MPCVD   (Microwave Plasma)
    ✓ RPCVD   (Remote Plasma)
    ✓ MOCVD   (Metal-Organic)
    ✓ AACVD   (Aerosol-Assisted)

Communication Protocols (3):
    ✓ SCPI/VISA  (Standard Commands for Programmable Instruments)
    ✓ OPC-UA     (OPC Unified Architecture)
    ✓ SECS-II    (SEMI Equipment Communications Standard)

Physics Models:
    ✓ Arrhenius deposition kinetics
    ✓ Film stress (intrinsic + thermal + gradient)
    ✓ Adhesion prediction (0-100 score)
    ✓ Thickness distribution (WIW/WTW uniformity)
    ✓ Conformality (step coverage)
    ✓ Optical properties (n, k, bandgap)
    ✓ Material properties (density, hardness, resistivity)
    ✓ Fault injection (temp spikes, leaks, upsets)

Integration:
    → Database: services/shared/db/models.py (Calibration)
    → CVD Models: services/analysis/app/models/cvd.py
    → CVD API: services/analysis/app/routers/cvd.py
    → Frontend: apps/web/src/components/manufacturing/CVDMES.tsx
